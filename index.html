ยังไม่มีทีมเข้าร่วมแข่งขัน กรุณาเพิ่มทีม

กรุณาสร้างหรือโหลดรายการแข่งขัน

ระบบจะทำการคำนวณโปรแกรมและจัดเรียงลำดับรอบแข่งขันให้อัตโนมัติ

ตารางคะแนนและสรุปผล (Standings)

อันดับ ชื่อทีม แข่ง ชนะ เสมอ แพ้ ได้ เสีย +/- คะแนน

ตารางคู่การแข่งขันแต่ละรอบ (Rounds)

* สามารถบันทึกสกอร์, สลับลำดับคู่ และกำหนดวันเวลาการแข่งได้ด้านล่าง

เพิ่มประเภทกีฬาใหม่

รายชื่อประเภทกีฬาและการกำหนดแต้มในระบบ

ประเภทกีฬา มีผลเสมอ? แต้มชนะ แต้มเสมอ แต้มแพ้ การจัดการ

คลังข้อมูลจัดการรายการแข่งขัน

เปิดดู แก้ไข หรือลบรายการการแข่งขันที่มีในฐานข้อมูล (เชื่อมโยงผ่าน Google Sheets หรือ Local Storage)

ไม่พบข้อมูลการแข่งขันที่เคยบันทึกไว้

ตั้งค่าการเชื่อมต่อ Google Sheets API

💡 วิธีใช้งานร่วมกับ GitHub Pages (วิธีที่ถูกต้อง):

1. นำ code.gs ไปวางใน Apps Script ของ Google Sheets แล้ว Deploy เป็น Web App (Execute as: Me / Access: Anyone)

2. คัดลอก Web App URL ที่ได้ไปวางด้านล่าง แล้วกด "บันทึกการเชื่อมต่อ"

⚠️ หลังแก้ไข code.gs ต้อง Deploy ใหม่ทุกครั้ง (New Deployment → New version) URL เดิมใช้ไม่ได้!
